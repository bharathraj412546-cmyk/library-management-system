# Library Management System

A professional, responsive, and dark-mode enabled Library Management System built with ReactJS.

## Features
- **Dashboard**: Overview of library statistics and recent transactions.
- **Books Management**: Full CRUD operations with search, category filtering, and pagination.
- **Users Management**: Manage library members with ease.
- **Issue/Return System**: Track book assignments, due dates, and return history.
- **Dark Mode**: Persisted theme toggle for comfortable viewing.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.

## Tech Stack
- ReactJS (Functional Components, Hooks)
- React Router DOM (Navigation)
- React Icons (Modern Iconography)
- Vanilla CSS (Custom Styling & Variables)
- Context API (State Management)
- LocalStorage (Persistence)

## Setup Instructions

### Prerequisites
- Node.js installed on your machine.
- A terminal (Command Prompt, PowerShell, or Git Bash).

### Installation
1. Navigate to the project directory:
   ```bash
   cd "Library management"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Project Structure
- `src/components`: Reusable UI components.
- `src/context`: State and Theme management.
- `src/pages`: Application pages.
- `src/styles`: Global styles and local component CSS.
